Project Overview

Rocky CIS Assurance Engine is a Bash-based compliance validation tool designed to assess Rocky Linux 8 systems — whether deployed as virtual machines or physical servers — against CIS benchmark security controls.

The solution enables security teams and system administrators to validate operating system hardening before production deployment by executing automated checks across critical security domains including filesystem configuration, services, network configuration, logging, authentication, and system file permissions.

🔹 What It Does (Clear and Direct)

Performs automated security configuration checks

Identifies misconfigurations and policy deviations

Produces structured PASS / FAIL / MANUAL results

Supports pre-deployment security assurance

Suitable for integration into DevSecOps workflows

🔹 Clean Value Statement

Provides structured security validation to ensure Rocky Linux systems are hardened and deployment-ready.
